# Praktikum Tuple Python

Program ini adalah aplikasi sederhana berbasis Python untuk mempelajari operasi dasar pada **tuple**. Program dibuat dengan menu interaktif di terminal.

---

## Fitur Program

Program ini menyediakan beberapa operasi dasar pada tuple:

1. **Menampilkan Tuple**
   - Menampilkan isi tuple dan jumlah elemennya

2. **Akses Elemen**
   - Mengambil elemen berdasarkan indeks

3. **Slicing Tuple**
   - Mengambil sebagian data dari tuple

4. **Gabung Tuple**
   - Menggabungkan tuple lama dengan data baru

5. **Ulang Tuple**
   - Mengulang isi tuple

6. **Ubah Tuple (via List)**
   - Menambah data
   - Mengubah data
   - Menghapus data  
   *(karena tuple bersifat immutable, maka diubah melalui list)*

7. **Keluar**
   - Menghentikan program
